# Profit Analysis Dashboard

## Project Overview

The Profit Analysis Dashboard is an interactive Power BI project designed to analyze business profitability, sales performance, and key financial metrics. It provides meaningful insights through interactive visualizations, helping businesses identify profit trends, evaluate performance, and make data-driven decisions.

## Objectives

- Analyze overall profit and sales performance.
- Monitor profit trends over time.
- Identify top-performing products or categories.
- Understand the key drivers affecting profit.
- Enable interactive business analysis using filters and slicers.

## Tools and Technologies

- Power BI
- Power Query
- DAX (Data Analysis Expressions)
- Data Modeling
- Interactive Visualizations

## Dashboard Features

- KPI cards displaying important business metrics
- Sales and profit trend analysis
- Profit distribution using donut charts
- Scatter plot for performance analysis
- Clustered bar and column charts
- Key Drivers Analysis
- Interactive slicers for filtering data
- Dynamic dashboard for business insights

## Dataset

The dataset contains business-related information such as:

- Sales
- Profit
- Product Category
- Sub-Category
- Region
- Customer Segment
- Order Date
- Quantity
- Discount

## Key Insights

- Compare profit performance across different categories.
- Identify products or regions generating the highest profit.
- Analyze sales trends over time.
- Discover factors that significantly influence profitability.
- Support strategic business planning with interactive analysis.

## Business Benefits

- Improves financial performance monitoring.
- Supports data-driven business decisions.
- Helps identify profitable business areas.
- Enables better resource allocation.
- Assists management in understanding profit drivers.
